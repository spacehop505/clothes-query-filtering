## Query Parameters
| Query Params  | Values |
| ------------- | ------------- |
| size=  | [s,m,l]  |
| material=  | [cotton, silk]  |
| dir=  | [desc, asc]  |
| order=  | [price]  |
| category=  | [1] category_id | 
| brand=  | [1] branding_id | 

## URI Structure
backend:
```
localhost:5000/clothing?size=m&material=cotton&dir=desc&order=price
```
client-frontend:
```
localhost:3000/clothing?size=m&material=cotton&dir=desc&order=price
```
